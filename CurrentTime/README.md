# CurrentTime.sh

### FILE:
CurrentTime.sh

### LANG:
BASH - Bourne Again Shell 

### DESC:
Returns the current time in 12hour format with numbers in white, colons in red, and either a yellow AM or a blue PM

### INST:
1. Copy sh file to local bin directory
2. execute:

```
CurrentTime
```
